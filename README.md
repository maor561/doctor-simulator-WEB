# Doctor Simulator — Israel Flight Simulation Community

Landing page for Doctor Simulator, the home of flight simulation enthusiasts in Israel.

## Features

- **News Board** — Aviation news and community updates (departures board style)
- **Community Tools** — Flight Target and Doctor Simulator Flights
- **WhatsApp Gates** — Four community WhatsApp groups (General, Buy/Sell, DCS World, CVFR)
- **Responsive Design** — Fully responsive, works on mobile and desktop
- **RTL Support** — Full Hebrew (right-to-left) layout support
- **Smooth Animations** — Flight path animations, ticker, and smooth transitions

## Tech Stack

- Pure HTML5
- CSS3 (Flexbox, Grid, Animations)
- No dependencies — static site

## Development

```bash
# Start a local dev server
npm run dev
```

Visit `http://localhost:3000` in your browser.
